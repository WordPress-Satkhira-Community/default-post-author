== Installation ==

= From within WordPress =
1. Visit 'Plugins > Add New'
2. Search for 'Default Post Author'
3. Activate Default Post Author from your Plugins page.
4. Go to Settings > General
5. At the bottom of that page you will see a dropdown with all your users list
4. That's all.

= Manual Installation =
1. Upload the `default-post-author` folder to the `/wp-content/plugins/` directory
2. Activate the Default Post Author plugin through the 'Plugins' menu in WordPress
3. That's all.


== Frequently Asked Questions ==

= Where is the settings page? =

Settings > General

Yes! It's compatible with both single sites and multisite.
